Clicks Protocol

Autonomous DeFi yield for AI agent treasuries on Base chain.

Every USDC payment your agent receives is auto-split: 80% liquid, 20% routed to the best DeFi yield via Aave V3 or Morpho. No lockup. No manual steps. No human required.

Your agent's treasury earns 7-10% APY while staying fully liquid. Three lines of code.

Why

AI agents earn USDC. That USDC sits idle in wallets — like cash in a taxi's glovebox overnight. Circle and Tether earn billions on this float. Your agent gets zero.

Clicks splits payments automatically. 80% stays liquid for your agent to use. 20% earns yield in DeFi. The protocol takes 2% of yield earned — never touches principal.

How It Works

Payment (100 USDC) │ ▼ ClicksSplitterV3 ├── 80 USDC → Agent Wallet (instant) └── 20 USDC → ClicksYieldRouter ├── Aave V3 (~7% APY) └── Morpho (~9% APY) └── auto-routes to best Withdrawal ├── Principal + Yield → Agent └── 2% of Yield → Protocol Treasury

Integration

Install

npm install @clicks-protocol/sdk ethers@^6

Quick Start

import { ClicksClient } from '@clicks-protocol/sdk';
import { ethers } from 'ethers';

const provider = new ethers.JsonRpcProvider('https://mainnet.base.org');
const signer = new ethers.Wallet(process.env.PRIVATE_KEY, provider);

const clicks = new ClicksClient(signer);

// Register your agent (one-time)
await clicks.registerAgent(agentAddress);

// Approve USDC spending (one-time)
await clicks.approveUSDC('max');

// Receive payment — auto-splits 80/20
await clicks.receivePayment('100', agentAddress);
// → 80 USDC liquid in agent wallet
// → 20 USDC earning yield in DeFi

// Later: withdraw everything
await clicks.withdrawYield(agentAddress);

SDK Methods

Write (require Signer)

  • registerAgent(agentAddress) — register agent, caller becomes operator
  • receivePayment(amount, agentAddress) — split USDC payment automatically
  • withdrawYield(agentAddress, amount?) — withdraw yield + principal
  • approveUSDC(amount) — approve splitter ("max" for unlimited)
  • setOperatorYieldPct(pct) — custom yield split (5-50%)
  • deregisterAgent(agentAddress) — remove agent registration

Read (work with Provider)

  • simulateSplit(amount, agentAddress) → { liquid, toYield, yieldPct }
  • getAgentInfo(agentAddress) → { isRegistered, operator, deposited, yieldPct }
  • getYieldInfo() → { activeProtocol, aaveAPY, morphoAPY, totalDeposited }
  • getFeeInfo() → { totalCollected, pending, treasury }
  • getOperatorAgents(operatorAddress) → string[]

Agent Referral Network (Multi-Level)

Recruit agents. Earn from their yield. Three levels deep. On-chain. Automatic.

You refer Agent B → you earn 40% of B's protocol fee Agent B refers Agent C → you earn 20% of C's fee Agent C refers Agent D → you earn 10% of D's fee Protocol fee = 2% of yield only. You earn from the fee, not from their principal. The referred agent pays nothing extra. Same yield. Same terms.

How to Refer

// Register a new agent with your address as referrer
await clicks.registerAgentWithReferral(newAgentAddress, yourAgentAddress);

// Or: the new agent registers themselves with your referral
await clicks.registerWithReferral(yourAgentAddress);

// Check your referral earnings
const stats = await clicks.getReferralStats(yourAgentAddress);
// → { directCount: 47, totalEarned: 2840000000n, claimable: 560000000n }

// Claim accumulated rewards
await clicks.claimReferralRewards();
