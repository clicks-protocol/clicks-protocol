Getting Started

Clicks Protocol is an on-chain yield layer for AI agents on Base. Your agent deposits USDC, and the protocol automatically splits it: 80% stays liquid, 20% earns yield.

Quick Start

Three lines to start earning yield:

import { ClicksClient } from '@clicks-protocol/sdk'
import { ethers } from 'ethers'

const provider = new ethers.JsonRpcProvider('https://mainnet.base.org')
const signer = new ethers.Wallet(process.env.PRIVATE_KEY!, provider)
const clicks = new ClicksClient(signer)

// Register agent + set split + deposit in one call
await clicks.quickStart('5000', agentAddress)

Installation

npm install @clicks-protocol/sdk

Requirements: Node.js 18+, ethers.js v6.

For MCP-compatible agents:

npm install @clicks-protocol/mcp-server

Write Methods

All write methods require a Signer.

registerAgent(agentAddress)

Register an AI agent. The caller becomes the operator.

await clicks.registerAgent('0xAgentAddress')

receivePayment(amount, agentAddress)

Deposit USDC. Automatically splits based on the agent's yield percentage (default 20%). Amount is human-readable USDC (e.g., "100" for 100 USDC).

await clicks.receivePayment('100', '0xAgentAddress')
// Splits: 80 USDC liquid, 20 USDC to yield

withdrawYield(agentAddress, amount?)

Withdraw accumulated yield + principal. Omit amount or pass "0" to withdraw everything. Only the agent, operator, or owner can withdraw. 2% protocol fee applies to yield earned (not principal).

// Withdraw everything
await clicks.withdrawYield('0xAgentAddress')

// Withdraw 50 USDC principal + accrued yield on that portion
await clicks.withdrawYield('0xAgentAddress', '50')

setOperatorYieldPct(pct)

Adjust the yield split for the calling operator (5-50%). Default is 20%. Pass 0 to revert to default.

await clicks.setOperatorYieldPct(30)
// Future deposits for this operator's agents: 70% liquid, 30% yield

approveUSDC(amount)

Approve USDC spending by the splitter contract. Pass 'max' for unlimited approval.

await clicks.approveUSDC('max')

quickStart(amount, agentAddress) Recommended

One function that does three things: register agent (if needed), set yield split, and deposit. Idempotent.

await clicks.quickStart('1000', agentAddress)
// Registers agent if not registered
// Sets 20% yield split
// Deposits 1000 USDC (800 liquid, 200 yield)

Read Methods

Work with both Signer and Provider.

getAgentInfo(agentAddress)

Returns registration status, operator, deposited principal, and yield percentage.

const info = await clicks.getAgentInfo('0xAgentAddress')
// { isRegistered: true, operator: '0x...', deposited: 20000000n, yieldPct: 20n }

getYieldInfo()

Returns active yield protocol (1=Aave, 2=Morpho), current APYs, total balance (principal+yield), and total deposited across all agents.

const yieldInfo = await clicks.getYieldInfo()
// { activeProtocol: 1, aaveAPY: 450n, morphoAPY: 380n, totalBalance: 1000000n, totalDeposited: 950000n }

simulateSplit(amount, agentAddress)

Preview how a deposit would be split without making a transaction. Returns liquid amount, yield amount, and effective yield percentage.

const preview = await clicks.simulateSplit('100', '0xAgentAddress')
// { liquid: 80000000n, toYield: 20000000n, yieldPct: 20n }

getYieldPct(agentAddress)

Get the effective yield percentage for an agent.

const pct = await clicks.getYieldPct('0xAgentAddress') // 20n

getOperatorAgents(operatorAddress)

Get all agents registered under an operator.

const agents = await clicks.getOperatorAgents('0xOperatorAddress')
// ['0xAgent1', '0xAgent2']

getFeeInfo()

Returns protocol fee stats: total collected, pending fees, treasury address, and fee basis points (200 = 2%).

const feeInfo = await clicks.getFeeInfo()
// { totalCollected: 1500000n, pendingFees: 50000n, treasury: '0x...', feeBps: 200n }

getUSDCBalance(address), getAllowance(address)

Check USDC balance and allowance for the splitter contract.

const balance = await clicks.getUSDCBalance('0xAddress')
const allowance = await clicks.getAllowance('0xAddress')

MCP Server Setup

npm install @clicks-protocol/mcp-server

Add to your MCP client config:

{
  "mcpServers": {
    "clicks": {
      "command": "npx",
      "args": ["@clicks-protocol/mcp-server"],
      "env": {
        "PRIVATE_KEY": "your-private-key",
        "RPC_URL": "https://mainnet.base.org"
      }
    }
  }
}

Contract Architecture

Five contracts work together:

Agent → ClicksSplitterV3 → 80% agent wallet
                         → 20% ClicksYieldRouter → Aave/Compound/Morpho
       ClicksRegistry (agent records)
       ClicksFee (protocol fee on yield only)

The protocol fee is taken from yield earned, never from deposits or principal. If an agent earns nothing, the protocol charges nothing.

Security

All contracts are verified on Basescan. No proxy upgrade patterns. No admin keys on user funds.

The protocol has not been audited by a third-party firm. A self-audit with Slither and DeFiHackLabs checklists is planned before public launch.
