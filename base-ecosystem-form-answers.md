## 3. Brief Project Description
Clicks Protocol is autonomous yield infrastructure for AI agents on Base. It helps agents earn yield on idle USDC from x402 payment flows while keeping 80% liquid for immediate use and routing 20% into yield strategies. Developers can integrate Clicks through Base-native smart contracts, a TypeScript SDK, and an MCP server for direct agent tooling.
