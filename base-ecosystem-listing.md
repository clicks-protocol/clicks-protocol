## Längere Beschreibung, 2 bis 3 Sätze
Clicks Protocol is autonomous yield infrastructure for AI agents on Base. It helps agents earn yield on idle USDC from x402 payment flows using an 80/20 split: 80% stays liquid for immediate use, 20% is routed to yield strategies. Clicks includes smart contracts on Base, a TypeScript SDK, and an MCP server for direct agent integration.
